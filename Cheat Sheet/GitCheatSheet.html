Configure Tooling

  • $git config - -global user.name "[name]"
    • Set a username for github
  • $git config - -global user.email "[name]"
    • Set a email for github

Create Repositories

  • $git init "[project name]"
    • Create LOCAL repository
  • $git clone "[url]"
    • Download an EXTERNAL repository

Make Changes

  • $git status
    • Lists all new/modified files to be commited
  • $git diff {"[1branch name]" "[2branch name]"}
    • Shows new files/changes not yet staged {or between branches}
  • $git add "[file name]"
    • Stages a new file to be cmmited
  • $git reset "[file name]
    • Unstages a file, preserving its contents
  • $git commit -m "[comment on commit]"
    • Commits file to current version history

Group Changes

  • $git branch
    • Lists all local branches in repo
  • $git branch "[branch name]"
    • Creates a new branch
  • $git checkout "[branch name]"
    • Switches to specified branch
  • $git merge "[branch name]"
    • Merges specified branch with current
  • $git branch -d "[branch name]"
    • Deletes specified branch

Refactor Filenames

  • $git rm "[filename]"
    • Deletes file from working directory and stages deletion
  • $git rm --cached "[filename]"
    • Removes file from version control but preserves it locally
  • $git mv "[original filename]" "[new filename]"
    • Changes filename and prepares for a commit

Review History

  • $git log
    • Lists version history for the current branch
  • $git log --follow "[filename]"
    • Lists version history for a file, including renames
  • $git show "[commit]"
    • Outputs metadata and changes of the specified commit

Redo Comits

  • $git reset "[commit]"
    • Undoes all commits after [commit], preserves changes locally
  • $git reset --hard "[commit]"
    • Discards all history/changes back to the specified commit

Sync Changes

  • $git fetch "[bookmark]"
    • Downloads history from repo bookmark
  • $git merge "[bookmark]"/"[branch]"
    • Combines bookmark's branch into current branch
  • $git push "[alias]" "[branch]"
    • Uploads all local branch commits to GitHub
  • $git pull
    • Downloads bookmark history and incorporates changes
